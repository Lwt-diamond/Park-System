# 停车场信息管理系统

#### 介绍
停车场信息管理系统
1.实现了管理用户的基本信息，办理业务（停车，取消停车），停车位统计以及价格和修改，打印停车账单等功能。帮助用户了解停车相关信息如价格、停车位 空闲个数的需求。
2.使用了Java事件驱动编程，JavaSwing，MVC设计模式，JDBC等技术，使用SQL Server作为数据库
